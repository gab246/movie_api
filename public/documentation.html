MyFlix


This movie web application allows users to access information about various movies and directors across multiple genres. Users are able to sign up to the application, update their personal details, as well as create a list of their favourite moives.

URl Endpoints and Request Methods documentation

Business Logic URL HTTP Method Parameters Request Body Data Format Response Body Data Format
Return a list of all movies to the user /movies GET - - a JSON object containing data about all the movies
Return data (description, genre, director, image URL) about a single movie according to title /movies/:title GET :title - a JSON object containing data about a single movie
Returns data about a genre by title /movies/genre/:genreTitle GET /:genreTitle - a JSON object containing data about a specific genre of a movie
Return data about a director (bio, birth year, death year) by name /director/:directorName GET :directorName - a JSON object containing data about specific director
Allow new users to register /users POST - a JSON object holding data about the user to add a JSON object containing data about the user that was added
Allow users to update their username information /users/:id PUT :id - a message indicating whether the username was updated
Allow users to add a movie to their list of favourite /movies/:id/:Title POST :id/:Title - a message indicating whether the movie was added to the list
Allow users to remove a movie from their list of favourites /movies/:id/:Title DELETE :id/:Title - a messaging indicating the movie was deleted from the list
Allow existing users to deregister /users/:id DELETE :id - a message indicating the user email has been removed
